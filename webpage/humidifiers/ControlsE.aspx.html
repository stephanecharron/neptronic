HRO - Wall Mount Humidity Controller

Details

Control Features
  • 6 humidification and dehumidification outputs
    • 4 analog outputs (0-10 or 2-10 Vdc)
    • 2 dry contacts
  • 4 Inputs
    • 2 Configurable analog inputs (0-10 or 2-10 Vdc)
    • Window or external temperature sensor input
    • Alarm status input
  • Selectable and configurable ramps
    • External humidity
    • High Limit
    • External demand low signal selector (LSS)
    • External humidity sensor averaging
    • Duct supply input
    • External demand signal
Operational Features
  • 24Vac operation
  • Adjustable setpoint with auto reset from external sensor
  • Backlit LCD with simple icon and text driven menus
  • Humidification and dehumidification indicator
  • Selectable Fahreinheit or Celsius scale
  • Multi level lockable access menu
  • Lockable setpoint and control mode
BACnet (HROB20 model only)
  • BACnet MS/TP @9600, 19200, 38400, or 76800bps
  • Selectable MAC address
  • Manual or automatic device instance configuration
  • Automatic Baud Rate Detection
